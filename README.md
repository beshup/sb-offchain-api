Helper off chain API for our smart contracts 

Startup:

**WINDOWS**
```
python -m venv env
.\venv\Scripts\activate
pip install -r requirements.txt  # Install all the libs.
python app.py
```

**LINUX/MACOS**
```
python -m venv env
source env/bin/activate
pip install -r requirements.txt  # Install all the libs.
python app.py
```
